Features
Everything you need for security assessment

From evidence collection to executive reports, BeGuardit covers the full assessment lifecycle.

🔎

10+ Collector Agents

OS info, network, services, packages, users, filesystem, AI runtimes, models, prompts, and RAG pipelines. Each runs independently with its own error boundary.

⚠️

Policy Evaluation Engine

Rule-based analysis with five severity levels. Customizable policy rules with remediation guidance built into every finding.

🔗

Attack Path Correlation

Graph-based analysis identifies multi-step attack chains that no single collector could detect alone.

📄

Multi-Format Reports

HTML, PDF, SARIF v2.1.0, and canonical JSON with SHA-256 integrity hashing. GitHub Code Scanning compatible.

🤖

AI Security Posture

Detect inference servers, audit deployed models, analyze prompt injection vectors, and evaluate RAG pipeline boundaries.

🏢

Enterprise Multi-Tenancy

Shared-schema PostgreSQL with row-level security, RBAC (admin/operator/viewer), Argon2id auth, and CSRF protection.

Get Started
Install in seconds

One command to install, one command to scan.

# Install the CLI globally
$ npm install -g @beguardit/cli
# Run your first assessment
$ beguardit start
# Or run the full platform with Docker
$ git clone https://github.com/benchbrex-USA/BEGUARDIT-CLI.git
$ cd BEGUARDIT-CLI && docker compose -f infra/docker-compose.yml up
# Check system prerequisites
$ beguardit doctor
